🚀 CÓMO EMPEZAR EN 3 PASOS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  CONFIGURAR AIRTABLE
   → Ve a https://airtable.com/api
   → Crea un Personal Access Token
   → Crea tabla "Users" con campos: Email, Password, Name, Status, CreatedAt

2️⃣  CONFIGURAR VARIABLES
   → cp .env.example .env.local
   → Edita .env.local con tus credenciales de Airtable

3️⃣  EJECUTAR
   → npm install
   → npm run dev
   → Abre http://localhost:3000/auth

⚠️  IMPORTANTE PARA PRODUCCIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

   Antes de desplegar, implementa:

   1. bcrypt - Hash de contraseñas
      npm install bcrypt

   2. jsonwebtoken - Tokens JWT reales
      npm install jsonwebtoken

   3. Rate limiting - Limitar intentos
   4. API Key en backend - No en cliente
   5. HTTPS - Conexión segura
   6. Cookies httpOnly - En lugar de localStorage

   Ver: AUTENTICACION.md (sección "Producción")
